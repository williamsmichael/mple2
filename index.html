Networker

Networker is a responsive web application for constructing private groups. The user group for this project is a “mock family network”. The idea is to reach within your network for assistance before seeking assistance outside of your network. The application provides opportunities to login into a secure environment, build a phone book, track user locations of residency, observe analytics based on the user group profiles, communicate through a project-based message system.
